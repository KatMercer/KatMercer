An example demonstrating the use of the <input type="color"> control.

Watch the colors change when you adjust the color pickers. As you make changes in the color pickers, the elements' color changes, as a preview (this uses the input event). When you close the color picker, the change event fires, and we detect that to change every paragraph to the selected color. This page may load slower than the bare page due to the increased code size required to make the color pickers work. Kat Mercer
